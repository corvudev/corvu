## Usage
<Code code={`
import Dialog from 'corvu/dialog'
// Or
// import { Root, Trigger, ... } from 'corvu/dialog'
`} lang="tsx" copyButton />

### Anatomy

<Code code={`
<Dialog.Root>
  <Dialog.Trigger />
  <Dialog.Portal>
    <Dialog.Overlay />
    <Dialog.Content>
      <Dialog.Close />
      <Dialog.Label />
      <Dialog.Description />
    </Dialog.Content>
  </Dialog.Portal>
</Dialog.Root>
`} lang="tsx" copyButton />

### Scrolling
By default, corvu disables scrolling on the body element  while the dialog is open. It will also add additional padding to the body to prevent the page from jumping when the scrollbar disappears. There is also a CSS variable called `--scrollbar-width` which is present when the scrollbar is hidden. You can use this to prevent any fixed element from jumping:

<Code code={`
<header
  class="fixed top-0 inset-x-0 z-50"
  style={{
    'padding-right': 'var(--scrollbar-width, 0)',
  }}
>
  Header
</header>
`} lang="tsx" copyButton />

### Nested dialogs
Feel free to nest dialogs however you want. Corvu is aware of nested dialogs and only dismisses the top-most dialog when you press the escape key or click outside the dialog.

## Accessibility
Adheres to the [Dialog WAI-ARIA design pattern](https://www.w3.org/WAI/ARIA/apg/patterns/dialog-modal) and [Alert Dialog WAI-ARIA design pattern](https://www.w3.org/WAI/ARIA/apg/patterns/alertdialog/).
