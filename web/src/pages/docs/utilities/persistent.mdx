# Persistent

Utility to create persistent components that keep their state and elements cached when removed from the DOM.

<PackageInfo packageName="solid-persistent" />

## Features
<Features features={[
  'Persists both JavaScript state and HTML elements',
  'Supports SSR',
]} />

## Installation

<Code code={`
  npm install solid-persistent
`} lang="bash" copyButton />

## Usage

<Code code={`
  import createPersistent from 'solid-persistent'
`} lang="tsx" copyButton />

<Code code={`
  const PersistedDialogContent = () => {
    const persistedContent = createPersistent(() => {
      return <input />
    })

    return (
      <Dialog>
        <Dialog.Trigger>
          Open
        </Dialog.Trigger>
        <Dialog.Portal>
          <Dialog.Content>
            {persistedContent()}
          </Dialog.Content>
        </Dialog.Portal>
      </Dialog>
    )
  }
`} lang="tsx" copyButton />
