# Prevent Scroll

Utility which prevents scrolling on the body element. Useful when working with modals, dropdowns, etc.

By default, this utility will add padding to the body element to prevent the page from jumping when the scrollbar is removed. This can be disabled by passing `preventScrollbarShift = false`.

It also adds the CSS variable `--scrollbar-width` to the body element, indicating the width of the currently removed scrollbar. You can use this variable to add padding to elements that are fixed, like a topbar for example.

## Usage
<Code code={`
import { createPreventScroll } from 'corvu'
`} lang="tsx" copyButton />
<Code code={`
const DialogContent: Component<{
  open: boolean
}> = (props) => {
  createPreventScroll({
    enabled: () => props.open
  })

  return (
    <Show when={props.open}>
      <div>Dialog</div>
    </Show>
  )
}
`} lang="tsx" copyButton />

Use the `--scrollbar-width` CSS variable to add padding to fixed elements to prevent the content from shifting when the scrollbar is removed:

<Code code={`
<header
  class="fixed top-0 inset-x-0 z-50"
  style={{
    'padding-right': 'var(--scrollbar-width, 0)',
  }}
>
  Header
</header>
`} lang="tsx" copyButton />
